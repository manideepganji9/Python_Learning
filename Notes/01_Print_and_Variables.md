# Print and Variables

## Print Function

Used to display output on the screen.

### Example 1

```python
print("Hello World")
```

Output:

Hello World

### Example 2

```python
print("My name is Manideep")
```

Output:

My name is Manideep

---

## Variables

A variable is a name given to a memory location that stores a value.

### Example

```python
name = "Manideep"
age = 17

print(name)
print(age)
```

Output:

Manideep
17

---

## Rules for Variable Names

✅ Can contain letters, numbers and underscore

```python
student_name = "Ram"
marks1 = 95
```

❌ Cannot start with a number

```python
1name = "Ram"
```

❌ Cannot contain spaces

```python
student name = "Ram"
```

---

## Python is Case Sensitive

```python
name = "Ram"
Name = "Shyam"

print(name)
print(Name)
```

Output:

Ram
Shyam
