# Data Types in Python

Python provides different data types.

## Integer (int)

Whole numbers.

```python
age = 17
```

## Float

Decimal numbers.

```python
height = 5.8
```

## String (str)

Text data.

```python
name = "Manideep"
```

## Boolean (bool)

Stores True or False.

```python
is_student = True
```

## None

Represents no value.

```python
x = None
```
